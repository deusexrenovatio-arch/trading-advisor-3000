# Documentation Index

## Scope of this repository
This repository contains the AI delivery shell for Trading Advisor 3000:
- development methodology,
- architecture governance,
- validation and gate policy,
- session and process lifecycle,
- durable process-state structure.

Business and trading logic are intentionally excluded.
