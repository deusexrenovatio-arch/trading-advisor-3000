# Repo-Local Skills

This directory is reserved for TA3000-specific skills only.

Use it for trading, product-plane data, research, or compute-runtime knowledge
that should not affect other repositories. Generic engineering workflows belong
in the global Codex skill root: `D:/CodexHome/skills`.

Each active repo-local skill must live in its own child directory with a
`SKILL.md` descriptor and pass the repo-local skill governance checks.

The generated mirror of active repo-local skills lives in
`docs/agent/skills-catalog.md`; update it with
`python scripts/sync_skills_catalog.py` after adding or changing a skill.
